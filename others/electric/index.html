电如何产生

发电方式

  • 火力发电
  • 风力发电
  • 水力发电
  • 核能发电
  • 太阳能发电
  • 地热发电
  • 海洋能发电（按能量的存储形式可以分为）
    • 机械能
      • 潮汐能（海水涨潮和落潮时形成的水的势能和动能）
      • 海流能/潮流能（海水流动的动能，主要指海底水道和海峡中较为稳定的流动，以及由于潮汐导致的有规律的海水流动）
      • 波浪能（海洋表面波浪所具有的动能和势能）
    • 热能
      • 海水温差能（表层海水和深层海水之间水温之差的热能）
    • 化学能
      • 海水盐差能（海水和淡水之间或两种含盐浓度不同的海水之间的电位差能）

电力的分类

  • 交流电：强度与方向都随着时间做周期性变化（我国50Hz）
  • 直流电：电流方向不随着时间做周期性变化

交流转直流叫整流，直流转交流为逆变

电力系统的组成

电力系统是指完成电能生产、输送、分配和消费的统一整体

电力流通的过程：发电 –> 输电 –> 配电 –> 用电

电的基本参数

基础知识

电流：单位时间里通过导体任一横截面的电量叫做电流强度，简称电流。通常用字母I表示，单位是安培
电压：单位电荷在静电场中由于电势不同所产生的能量差的物理量。其大小等于单位正电荷因受电场力作用从A点移动到B点所做的功
常用电压等级：
高压：1kV以上的电压。常用的有10kV、35kV、110kV、220kV
超高压：330kV、500kV、750kV
特高压：1000kV以上交流电压，±800kV的直流电压
功率：物体在单位时间内所做的功的多少，即功率是描述做功快慢的物理量。单位是J/S，焦耳每秒，国际单位为W

电量单位

发电厂装机容量单位：千瓦(kW)。其他常用单位：万千瓦、兆瓦、亿千瓦
常用单位转换：
1千瓦 = 1000瓦，1兆瓦 = 1000000瓦，1兆瓦 = 1000千瓦，1万千瓦 = 10000000瓦，1万千瓦 = 10兆瓦
1度电 = 1千瓦时
装机容量85万千瓦的机组，满负荷运行1个小时，发电量 = 85万千瓦时 = 85度电

电量的计算

一台机组不可能一直以一个恒定的输出功率发电，如何计算在指定时间内机组的发电量？

积分法计算

原理：电量单位为千瓦每小时，将1个小时平均分成12份，以5分钟为一个间隔，测量出每个5分钟时刻点上的功率，单位为千瓦，数值分别为P1、P2、……、P12，则该小时内的总电量 = P1千瓦*(1/12小时)+P2千瓦*(1/12小时)+……+P12千瓦*(1/12小时)

电力市场常见名词

两级市场：国家电网公司、南方电网公司
三级调度：国网调度、区域调度、省级调度
五大发电集团：华能、大唐、华电、国电（中国国电集团有限公司）、中国电力投资集团
区域发电集团：鲁能、浙能……

附：电力系统与各大公司之间的关系图：

发电量：发电机产生的电量
上网电量：卖给电力公司的电力（也就是最终流入电力公司的电量）
用电量：用户使用的电量
计量：对发电量、上网电量、用电量等的计算
计量点：一般设在用电侧
交易组织方式：集中、挂牌、双边
交易周期：年度、多月、月度、多日、日前、日内
线损：电网电能损耗的简称。是电能从发电厂传输到电力用户过程中在输电、变电、配电盒营销各环节中所产生的电能损耗和损失（线损包含变损）
变损：变压器损耗
同期线损：线损计算中供电量和售电量使用同一时刻电量的计算方法
CT：（Current Transformer）电流互感器。起电流变换和电气隔离作用
PT：（Potential transformer）电压互感器。用来变换线路上的电压的仪器
绕组：构成与变压器标注的某一电压值相对应的电气线路的一组线匝。各个副绕组的匝数不同，则其端电压也不同，因此多绕组变压器可以向几个不同电压的用电设备供电。在电力系统中最常用的是三绕组变压器
T接线路：指从甲方向乙方供电的线路中间接出一条线路，向第三方丙供电。相当于一条线路带了乙方和丙方两个用户
Π接线路：指将原来从甲变电站至乙变电站的一条输电线路，开断后形成 Π接接入丙变电站，线路Π接后将原来一条单独的输电线路开断形成两条独立的输电线路

电网等级

  • 低压

    220 ~ 380V。一般家用

  • 中压

    10kV ~ 35kV。配电网电压

  • 高压

    35 ~ 220kV

  • 超高压

    330（包含） ~ 1000kV 以下

  • 特高压

    1000kV 及以上交流电网或 ±800kV 及以上直流电网

电力系统的实时平衡

由于电力的物理特性，电力生产和消耗必须保持严格的实时平衡。即发电功率 = 用电功率 + 网损。当平衡被打破时，电能质量会受到影响（频率大幅波动等），一般要求机组在3s内做出反应（一次调频响应要求）

一次调频：电网频率标准为50Hz，频率偏差不得超过±0.2Hz，当频率超出规定的正常范围后，通过汽轮发电机组本身调节系统自动调整汽轮机出力水平，修正电网频率波动
二次调频：一次调频只能缓和电网频率的改变程度，但不能维持频率不变，此时需要通过人为或系统自动控制增减某些机组的负荷，保持电网频率在合理范围内，这个过程称为二次调频

电能量计量系统

什么是电能量计量系统

TMR(tele-meter reading system)：对电能量数据进行自动采集、远传和存储、预处理、统计分析，以支持电力市场的运营、电费结算、辅助服务费用结算和经济补偿计算等

功能

  • 应用系统支撑平台：主要包括数据集采、数据检查及处理、数据存储、系统状态监视、事件告警及记录、数据库管理、系统安全管理、通用图形系统、通用报表系统等
  • 系统的应用功能：包括电能计量管理、统计分析、系统维护、web浏览等

关键字

RTU定义

包含采集终端的相关参数的集合，包含厂站计量设备配置信息、数据存储信息、通信规约等模型的映像参数

  • 终端链路地址
  • 采集通道集合关系（专线、网络、电话通道）
  • 积分周期
  • 存储周期
  • 测量点数据类型定义

费率协议

包含日时段、费率、有效周期、特殊日信息的集合，一般包含以下元素：

  • 费率名称定义
  • 日费率过程
  • 周费率过程
  • 季节定义
  • 有效周期
  • 特殊日列表

负荷曲线数据

电能量计量系统定期把每个计量设备的数据定期采集处理存储后形成的一个反映时域电能量数据变化的额数据

记账数据

电能量计量随机电能表面板显示数据的快照的集合

以下数据在电表中以寄存器方式存储，记账数据至少包含以下数据：

  • 电表号
  • 日期时间
  • 当前电能量数据
  • 当前分时电能量数据
  • 月冻结总电量
  • 月冻结分时电能量数据
  • 当前分时需量数据
  • 月冻结分时需量

随机事件

计量表计、电表采集处理终端随机产生的事件数据，记录厂站设备异常的信息。

例子：

  • 电表参数设置
  • 电表时间设置
  • 电表缺相
  • 电表采集处理终端参数被修改
  • 电表采集处理终端时间被修改
  • 电能表表盖被打开
  • 断电
  • 上电

积分周期

负荷曲线存储的间隔
一般负荷曲线的积分周期是60分钟的约数，一般用于计量系统的数据积分周期为15分钟、5分钟、最小1分钟

通信规约

约定了主站、电能量采集处理终端、电能计量表计之间的接口、硬件、数据结构数据传送协议

主站规约：

  • DL/T719
  • SCTM
  • DLMS
    表计规约：
  • IEC 870-5-102
  • DL645
  • DLMS
  • 各电表厂家规约

统计数据

电能量计量系统把存储的数据按照内部计算规则定义的二次处理数据

自动进程

电能量计量系统中预先定义的周期性的处理定义，包括定期的数据采集、统计数据生成、报表生产、系统日常事务处理

  • 同时性

  • 安全性
