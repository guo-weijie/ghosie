指令

指令基础

上述指令说明如下：

  • userName：当前登录账户的用户名
  • deviceName：设备名称
  • ~：当前所在的目录
  • $：提示字符。（root账户的提示字符为 #，普通用户的提示字符为 $）
  • command：指令名称
  • [-options]：中括号不是指令，而是表示这个指令是一个选项指令。实际指令中，通常会带 - 号，例如 -h，有时候会使用选项指令的完整全名，则选项前会带 – 号，如 --help
  • parameter1 parameter2：选项或者 command 的参数
  • 指令、选项、参数等中间以空格来区分，不论空几格都视为一格
  • 指令太长的时候，可以使用反斜杠 \ 来使指令换行（虽然换行，但指令还是连续的）

常用的热键

Tab

  • 接在一串指令的第一个词的后面，为 命令补全
  • 接在一串指令的第二个词以后时，为 文件补齐
  • 若安装 bash-completion，在某些指令后面使用 tab 时，为 选项/参数的补齐
[dmtsai@study ~]$ date --[tab][tab] <==[tab]按键是紧接在 -- 后面！
--date --help --reference= --rfc-3339= --universal
--date= --iso-8601 --rfc-2822 --set= --version
# 瞧！系统会列出来 date 这个指令可以使用的选项有哪些喔～包括未来会用到的 --date 等项目
